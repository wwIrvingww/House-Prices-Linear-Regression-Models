En otras palabras, despus de varias iteraciones, el mtodo no pudo encontrar un conjunto de parmetros estables. Esta situacin puede ocurrir cuando hay variables altamente correlacionadas (multicolinealidad) o cuando se presenta una separacin completa o casi completa de las clases en algunos predictores.

Otro aviso relevante es prediction from rank-deficient fit; attr(*, non-estim) has doubtful cases. Este mensaje sugiere que el modelo es deficiente en rango lo que significa que existen redundancias entre las variables predictoras, impidiendo que algunos coeficientes se estimen de manera nica. Esto suele suceder cuando hay variables con alta correlacin o cuando se incluyen ms predictores de los que la informacin en la muestra puede soportar.
